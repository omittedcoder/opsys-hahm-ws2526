Permissions
===========

## Calculation

Letter | Permissinos | Value
-------+-------------+------
 r     | read        | 4
 w     | write       | 2
 x     | execute     | 1
 -     | none        | 0

## Commands

- whoami: current user
- chmod <0-7|u+-rwx><0-7|g+-rwx><0-7|o+-rwx> <file>: change permissions
